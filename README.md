# Password_generator
The Random Password Generator is a command-line Python application that creates secure and customizable passwords based on user preferences.
 
The user can specify:

Password length
Whether to include letters
Whether to include numbers
Whether to include symbols

By allowing these options, the program generates strong and unpredictable passwords, enhancing security for online accounts and systems. The program demonstrates several essential programming concepts such as functions, conditional statements, loops, exception handling, user input validation, and the use of built-in Python modules (random and string).

Conclusion

The Random Password Generator – Command Line Version is a simple yet powerful Python application that demonstrates essential programming concepts such as functions, conditional logic, loops, and exception handling. By allowing users to customize password characteristics, the program generates secure and unpredictable passwords that enhance digital security.
